## 📝 Description
*Hostel Sync is a centralized digital platform designed to streamline and modernize hostel operations by replacing manual registers and scattered communication systems*

## 🛠️ Tech Stack
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MondoDB
* **Tools:** Github, VS Code, HP Victus Laptop

## 🌟 Key Features
* **Feature 1:** Digital gate pass system
* **Feature 2:** parent notification
* **Feature 3:** complaint Management with SLA
*  **Features 4:** Mess Menu Poll
